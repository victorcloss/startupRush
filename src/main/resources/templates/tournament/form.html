Configure um novo torneio para competição entre startups

Como funciona um torneio

O torneio é organizado em rodadas eliminatórias, onde startups competem em batalhas diretas. Cada batalha é decidida pela pontuação das startups, com eventos que podem ser aplicados durante as disputas para alterar os pontos.

Requisitos:
  • Mínimo de 4 startups participantes
  • Número de startups deve ser uma potência de 2 (4, 8, 16, etc.)
  • Cada startup inicia com 70 pontos

Novo Torneio

Ao criar um novo torneio, você será redirecionado para selecionar as startups participantes. Após selecionar as startups, você poderá iniciar as batalhas!
